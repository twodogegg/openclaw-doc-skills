# openclaw-docs-skills

OpenClaw 文档类 skills 合集（中文优先，英文回退）。

## 包含技能

- openclaw-cli
- openclaw-gateway
- openclaw-tools
- openclaw-channels
- openclaw-providers
- openclaw-platforms
- openclaw-concepts
- openclaw-help
- openclaw-install

## 在其他电脑安装

1. 克隆仓库到任意目录。
2. 进入仓库执行：

```bash
bash install.sh
```

脚本会把本仓库里的 `openclaw-*` 目录同步到 `~/.codex/skills/`。

## 更新方式

在新电脑拉取最新代码后再次执行：

```bash
bash install.sh
```
